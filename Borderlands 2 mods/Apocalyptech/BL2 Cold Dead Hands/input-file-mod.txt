    # Enemies will always drop the exact gun/shield that they were using, and
    # weapons/shields are in general not otherwise findable in the world (treasure chests,
    # for instance, will not contain guns or shields).
    #
    # In the default configuration, enemy gear is MUCH improved from stock -- you will be
    # fighting enemies with gemstones, E-Tech, and even legendaries, across all weapon
    # types.  Various options for controlling the quality of gear used by enemies (and
    # thus available to yourself) can be found below.  See the README file for full
    # details.

    #<Main Mod (best to leave all this checked)>

        #<Early Game Gear Unlocks><lock>

            # This is a copy of UCP's Early Game Unlocks, which allows various weapon and
            # shield levels from the very beginning of the game.  Without this in place,
            # early-game enemies can spawn without ANY equipped gear, which is weird.

            set GD_Itempools.Scheduling.Gamestage_02:ConstantAttributeValueResolver_0 ConstantValue 1

            set GD_Itempools.Scheduling.Gamestage_03:ConstantAttributeValueResolver_0 ConstantValue 1

            set GD_Itempools.Scheduling.Gamestage_07:ConstantAttributeValueResolver_0 ConstantValue 1

        #</Early Game Gear Unlocks>

        #<Skinpool Setup><lock>

            # This section is copied directly from UCP's Skinpool Reassignments section,
            # which frees up some 600 skin/head pools for use in mods.  I'm including it
            # in here so that no part of UCP is actually required, and this mod is
            # theoretically able to function completely standalone.

{skinpool_setup}

        #</Skinpool Setup>

        #<Disable World Weapon/Shield Drops>

            # This section disables world drops for weapons and shields, so those items will
            # only drop from what the enemies are carrying.  Rather than deleting/disabling
            # the items entirely, we're changing them to draw from an ammo/money pool so that
            # the rest of the probabilities in the pools aren't affected.
            #
            # Note that this doesn't *completely* get rid of world weapon/shield drops, but it
            # should get rid of the vast majority of them.
